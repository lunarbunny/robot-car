1. run pytoC.py
2. put generated conversion.txt in c directory
3. run mazefinder_final.c (remember to set maze size to x * y)

all.txt is generated by maze_test.py: contains random maze lists
csvify_maze.py creates maze.csv from a string in all.txt
pytoC.py uses maze.csv to generate conversion.txt
conversion.txt used by mazefinder_final.c for final algo

maze_test.py is the brute force tester, maze.py can be used to show specific route for specific maze
